escalier by kydos

Scala API for DDS

Escalier is a thin Scala wrapper around the OpenSplice DDS Java API. This is mostly an experiment to understand how we might design a Scala API for DDS. Yet, this API is fully functional and provides the same level of performance as the OpenSplice DDS Java API!

Dependencies

To be able to use Escalier you need to have installed the following software on your machine: - OpenSplice DDS >= v5.x (http://www.opensplice.org) - Simple Build Tool (SBT) (http://code.google.com/p/simple-build-tool/)

Install

Make sure that SBT and OpenSplice DDS are properly installed and configured. At this point you should do the following, from the Escalier home directory: $ ./configure $ sbt update $ sbt compile $ sbt package $ sbt doc
